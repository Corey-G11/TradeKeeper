# TradeKeeper — Promo Video Script (built for Google Gemini / Veo, 8-sec clips)

A ~60-second vertical ad for TradeKeeper, broken into **8 clips of 8 seconds**.
Each clip is a **standalone Veo prompt** — Veo doesn't remember previous
generations, so every prompt repeats the same **Style Anchor** to keep the look,
character, phone, and colors consistent across clips. Generate each clip in the
Gemini app, then stitch in order.

---

## How to use this (read once)

1. **In the Gemini app → Video (Veo).** Set output to **9:16 (vertical)**.
2. For each clip below, paste **`STYLE ANCHOR` + the clip's `PROMPT`** as one
   prompt. Generate. If a clip looks off, regenerate (cheap) — don't fight it.
3. **Consistency trick (important):** generate **Clip 1 first**, take a clean
   still frame from its end, and use Veo's **image-to-video** (“upload image”)
   as the starting frame for the next clip. Repeat. This keeps the same trader +
   phone + room across all 8 clips far better than text alone.
4. **Audio:** generate clips with **ambient + SFX only** (prompts are written
   that way). Do **not** rely on Veo to speak the voiceover — the narrator voice
   won't match across clips. Instead, in your editor lay down **one** continuous
   music track + **one** voiceover recording (script in the `VO` lines).
5. **Stitch** in CapCut / Premiere / DaVinci: clips in order, hard cuts on the
   beat, add the VO + music, then drop your **real screen recordings** over the
   "[SCREEN]" clips for crisp UI (optional but recommended).
6. Export **9:16, 1080×1920, 30fps**. Total ≈ 64s (trim to ~45–60s for ads).

> Subject = a launch ad for **TradeKeeper** (your TradeZella-style trading
> journal). To retarget, only the VO + the "[SCREEN]" overlays change.

## 🎨 STYLE ANCHOR — prepend this to EVERY clip prompt

```
Style: cinematic, high-end fintech ad, vertical 9:16, shallow depth of field,
crisp 4k, moody. Color palette: near-black navy background (#070b14), neon
mint-green (#1ed98a), electric purple (#8d7bff), cyan accents. Subject: the
same focused male day trader, late 20s, short dark hair, plain dark hoodie,
in a modern dark home-trading room with soft RGB rim light and a glowing
monitor bokeh behind him, holding the same matte-black smartphone. Clean,
premium, energetic but not cheesy. Subtle film grain, smooth gimbal motion.
Negative: no text artifacts, no warped hands, no extra fingers, no logos,
no watermark, no cartoon look.
```

## Tips for flawless stitching
- **One music track** across all clips = instant cohesion; cut clips on the beat.
- Keep **camera motion direction consistent** (mostly push-ins) so cuts feel
  intentional.
- If a clip's trader looks different, reroll using the **image-to-video** anchor
  from the previous clip's last frame.
- For the "[SCREEN]" beats, a 1–2s **real screen recording** of the actual app
  overlaid on the phone sells it better than any AI UI.
- 8 clips = 64s. For TikTok/Reels/Shorts, cut to **~45s** by dropping Clip 2 or 7.
